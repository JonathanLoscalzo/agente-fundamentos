Simple cleaning robot - Neuronal network

El robot: limpia si está sucio, sino se dirige hacía la derecha (o izquierda), luego cambia de dirección cuando llega al final.

Se está utilizando una red neuronal precargada para realizar la operación.

Tiene una predisposición a avanzar según de cual ubicación proviene, y en el caso de limpiar, vuelve hacía atrás.

Seguramente no sea la forma mas eficiente para realizar el agente, además que no hay ninguna medida de rendimiento / recompompensa que vaya agregando más datos a la red.
